## ACCIO: ACcelerated Compact IO

Provides binary streaming IO using accelerating technology and zlib compression

### Installation

```
mkdir build
cd build
cmake [-DOPTIONS=ON/OFF] ../
make install
```

where OPTIONS can be:

- BUILD_TESTING: enable testing (values=ON/OFF)
